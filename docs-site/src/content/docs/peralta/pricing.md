## Retainer Options (Post-Engagement)

### Tier 1 — Watch ($3,500/month)

- Monthly dependency audit and CVE monitoring
- Priority access (10h/month)
- Quarterly threat model refresh
- New contract review (up to 300 SLOC/month)

### Tier 2 — Guard ($7,500/month)

Everything in Watch, plus:

- Continuous on-chain transaction monitoring
- Wallet anomaly detection with alerting
- Monthly platform scan
- New contract review (up to 1,000 SLOC/month)
- 24-hour incident response SLA

### Tier 3 — Shield ($15,000/month)

Everything in Guard, plus:

- Dedicated VLN analyst (40h/month)
- 4-hour incident response SLA
- Unlimited smart contract review
- Quarterly penetration test (40h)
- Monthly executive report
